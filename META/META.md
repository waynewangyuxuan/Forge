# META - Index

This is the root documentation folder for the Forge project. It contains all specifications, architecture documents, and progress tracking.

## Purpose
Central documentation hub for Forge - an Electron app that uses AI to generate complete runnable projects from natural language specs.

## Folders

| Folder | Purpose |
|--------|---------|
| [CORE/](CORE/) | Product specification and development principles |
| [FRONTEND/](FRONTEND/) | Frontend architecture - state management, pages, components, design system |
| [BACKEND/](BACKEND/) | Backend architecture - layered design, domain logic, data models, config-driven patterns |
| [IPC/](IPC/) | IPC layer - communication protocol between frontend and backend |
| [MILESTONES/](MILESTONES/) | Development milestones - M0 to M9, incremental delivery plan |

## Files

| File | Purpose |
|------|---------|
| [FILE-STRUCTURE.md](FILE-STRUCTURE.md) | **Single source of truth** for all file/directory structures |
| [PROGRESS.md](PROGRESS.md) | Progress tracker - append-only log of completed work by date |
| [TODO.md](TODO.md) | Todo tracker - append-only log of upcoming work by date |
