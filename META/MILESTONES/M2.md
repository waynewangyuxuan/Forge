## IPC 通道

| 通道 | 说明 |
|------|------|
| `project:list` | 获取项目列表 |
| `project:create` | 创建项目 |
| `project:get` | 获取项目详情 |
| `project:archive` | 归档项目 |
| `project:delete` | 删除项目 |
| `version:list` | 获取版本列表 |
| `version:create` | 创建版本 |
| `version:get` | 获取版本详情 |
| `version:setActive` | 设置活跃版本 |
