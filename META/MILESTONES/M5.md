## TODO.md 格式

Forge 生成的 scaffold 采用两层结构：

### TODO.md（轻量索引）

```markdown
# TODO

> Project: my-project
> Generated: 2024-01-01T00:00:00.000Z

## M1: Setup
- [ ] 001. Initialize project
- [x] 002. Add dependencies

## M2: Core Features
- [ ] 003. Implement user module
```

### MILESTONES/M1-setup.md（详细任务）

```markdown
# M1: Setup

> Project infrastructure and tooling

## Tasks

### 001. Initialize project

**Description:**
Create the basic project structure with package.json

**Verification:**
package.json exists and contains correct metadata

**Depends:** none

---

### 002. Add dependencies

**Description:**
Install TypeScript, ESLint, and other dev dependencies

**Verification:**
node_modules populated, no npm errors

**Depends:** 001

---
```

### 解析结果

```typescript
interface ExecutionPlan {
  milestones: Milestone[]
  totalTasks: number
  completedTasks: number
}

interface Milestone {
  id: string            // "M1"
  name: string          // "Setup"
  description?: string  // from milestone file
  tasks: Task[]
}

interface Task {
  id: string            // "001"
  title: string         // "Initialize project"
  description?: string  // from milestone file
  verification?: string // from milestone file
  depends: string[]     // ["001"]
  status: TaskStatus    // 'pending' | 'completed'
  milestoneId: string   // "M1"
}
```
